* `npm install` 安装依赖

* 打开CMD，用mongod --dbpath=xxx(你存放此项目的文件夹地址)\db --port=27018

* 运行app.js
 
* 打开浏览器，转向  [localhost:8081](http://localhost:8081/ "Title")
